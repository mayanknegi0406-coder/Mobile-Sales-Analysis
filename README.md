# 📊 Mobile Sales Analysis (Power BI)

This project showcases an interactive **Mobile Sales Dashboard** built using Power BI to analyze sales performance across different dimensions such as cities, brands, models, and payment methods.

## 🔍 Key Insights
- **Total Sales:** 769M+
- **Total Quantity Sold:** 19K units
- **Transactions:** 4K+
- **Average Price:** ~40K per unit
- **Top City:** Delhi
- **Top Brands:** Apple, Samsung, OnePlus
- **Top Models:** iPhone SE, OnePlus Nord, Galaxy Note 20
- Sales are consistent across months with minor fluctuations
- Payment methods (UPI, Debit Card, Credit Card, Cash) are almost equally used

## 📈 Features
- Month-wise and day-wise sales trends
- Brand and model performance comparison
- City-wise sales distribution
- Customer ratings analysis
- MTD (Month-To-Date) and Year-over-Year comparison

## 🛠 Tools Used
- Power BI  
- Data Analysis  
- Data Visualization  

## 🎯 Purpose
The goal of this project is to understand sales trends, customer behavior, and overall business performance, making it useful for data-driven decision-making and showcasing data analytics skills.
